Marketing Nurture Sequence (Cron: daily at 10am UTC)
Cron Daily Nurture Sequence (Multi-Step)
Subject from each step's template (MKTG_STEP_1, MKTG_STEP_2 …)
TriggerMarketing Consent = TRUE, Free/no plan, not yet responded or unsubscribed, last send ≥ N days ago
StagePre-membership
ToEligible free-tier travelers
ContentDrip sequence — steps stop when traveler clicks CTA or upgrades
TrackingMarketing Communications table logs each send + response
api/marketing.js › cron action
Email provider: Resend (api.resend.com) · From: TravelForGrowth@transformedbytravels.com
SMS provider: Twilio (journal daily only, requires phone in Traveler record)
Templates: Stored in Airtable "Email Templates" table, keyed by Code field
Cron schedules: Defined in vercel.json
